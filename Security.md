# Security Policy

## Supported Versions
Only the latest version of this pipeline is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
We take the security of operational systems seriously. If you discover a vulnerability (e.g., exposed API keys, insecure dependencies, or data leakage), please do **not** open a public issue.

### Reporting Protocol
1.  **Email:** Contact the maintainer directly at `me@sourav.ru`.
2.  **Details:** Please include steps to reproduce the vulnerability.
3.  **Response:** We will acknowledge receipt within 48 hours and provide a timeline for a patch.

Thank you for practicing responsible disclosure.
